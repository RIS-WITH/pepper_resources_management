# Pepper resources management

[![Dependency Status][Dependency-Image]][Dependency-Url]

This is a meta package grouping necessary components to manage physical resources of the Pepper robot.

It is composed of:

 - A **head** manager
 - An **arm** manager
 - A **base** manager
 - A synchronizer
